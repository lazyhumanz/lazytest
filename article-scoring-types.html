Understanding Scorecard Types

How it works: Start with a perfect score (100%) and deduct points for each error found during the audit.

Key Characteristics:

  • All parameters are of type "Error"
  • Field type must be "Counter" (errors can occur multiple times)
  • Cannot use "Radio" (Yes/No) fields
  • Score decreases with each error instance
  • Final score = 100 - (sum of all error deductions)

Example:

If an audit has 2 instances of "Grammatical Errors" (5 points each) and 1 instance of "Lack of Empathy" (10 points), the final score would be: 100 - (5 + 5 + 10) = 80%

Important:

If a "Critical Fail Error" (50 points) occurs, the audit automatically fails regardless of other scores. These are marked with "Auto-Fail" flag.

How it works: Start at zero and earn points for each achievement demonstrated during the audit.

Key Characteristics:

  • All parameters are of type "Achievement"
  • Field type must be "Radio" (Yes/No) - achievements are binary
  • Cannot use "Counter" fields
  • Cannot have "Error" type parameters
  • Score increases with each achievement
  • Final score = sum of all achievement points

Example:

If an audit demonstrates "Excellent Empathy" (10 points), "Proactive Solution" (10 points), and "Clear Communication" (5 points), the final score would be: 0 + 10 + 10 + 5 = 25%

How it works: Combines both deductive and additive scoring. Deduct points for errors and add points for achievements.

Key Characteristics:

  • Can have both "Error" and "Achievement" parameters
  • Can have "Bonus" parameters for exceptional performance
  • Error parameters use "Counter" field type
  • Achievement parameters use "Radio" field type
  • Starting score can be configured (typically 100)
  • Can allow scores over 100% if configured

Example:

Starting at 100%, if an audit has 1 "Grammatical Error" (-5 points) but demonstrates "Excellent Empathy" (+10 points) and a "Proactive Solution" (+10 points), the final score would be: 100 - 5 + 10 + 10 = 115% (if over-100 is allowed)

Configuration Options:

  • Max Bonus Points: Limit on total bonus points that can be awarded
  • Allow Over 100%: Whether scores can exceed 100%
